Setup Instructions
🔥 Firebase Setup (5 minutes)
1. Go to console.firebase.google.com
2. Click "Add Project" → name it hkslaw-os
3. Enable Firestore Database (test mode to start)
4. Go to Project Settings → Your apps → Web app
5. Copy the config values into the fields above
6. All task completions, KPIs, partners sync instantly

✓ Free tier: 1GB storage, 50K reads/day
📊 Google Sheets Setup (3 minutes)
1. Create a new Google Sheet at sheets.google.com
2. Name it HKS Law — Operating Data
3. Share it with your email + "Editor" access
4. Go to console.cloud.google.com
5. Enable Sheets API → Create API Key
6. Copy Sheet ID from URL and API key above

✓ Weekly KPIs, partners, tasks all export automatically
